Perks Killers Must Never Use

Devour Hope

Talk about getting carried by overpowered perks. Does one even require any skill to "win" with this? After five hooks, survivors can be eliminated from the game through mori even if they have never been hooked before. The only goodside is that it prevents camping. But if the killer was actually good, they would not need to camp in the first place.

Franklin's Demise

The one and only purpose of this perk is to annoy survivors.

No One Escapes Death

This perk punishes survivors for finishing all the generators because the killer did not know how to pressure the map effectively.
